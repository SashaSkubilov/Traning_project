# Training Program System

Проект представляет собой REST-сервис на Spring Boot для управления тренировочными программами.
Логика упражнений перенесена из консольного приложения на C++.

## Основной функционал
* **Учет упражнений:** поддержка силовых (вес/повторы), кардио (дистанция/темп) и периодов отдыха.
* **REST API:**
    * Получение данных об упражнении по его ID.
    * Фильтрация упражнений по целевой группе мышц.

## Технологический стек
* Java 17 / Spring Boot 3
* Spring Data JPA (работа с базой данных)
* H2 Database (хранение данных в оперативной памяти)
* Lombok (сокращение шаблонного кода)

## Как запустить приложение
1. Откройте проект в IntelliJ IDEA.
2. Дождитесь загрузки зависимостей Maven.
3. Запустите класс `TrainingProgramApplication`.
4. API будет доступно по адресу `http://localhost:8080/api/exercises`.
